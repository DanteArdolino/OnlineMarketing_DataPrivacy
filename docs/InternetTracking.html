How You Are Tracked Online

Online advertising, inescapable due to the wealth of data readily available on each and every user. A single site can contain trackers that are hidden that take the smallest bits of data.

Operators of search engine DuckDuckGo want to make “do not track” mode a mandatory standard. Companies and websites still continue to keep data profiles on us even if we have opted out. Most common tools to collect data are Cookies which are tiny files placed on the computer when visiting a website. 2 basic types of cookies session and persistent. Session which are good enable us to move from one page to another without having to login at every page. Should automatically be deleted when leaving the website or “session”

Persistent cookies can be the bad cookies as they are placed in the computer and stay in the computer. They are used primarily by marketing firms to track browsing history, for example if you visit a sports store, you will see more ads for other sports stores which can become intrusive. Another method is browser fingerprinting, a website can send a java code to run locally on your computer. This code collects snippets of information which the code creates known as the browser fingerprint. This can be used to track you when you re-enter the website or even across multiple websites.
